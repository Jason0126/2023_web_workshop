# 2023_web_workshop
此專案為2023叡揚資訊自主學習課程的課堂作業，主要是將老師提供的檔案進行內容的修改和新增。

## 介紹
此網頁提供了書籍的資料顯示、查詢、新增以及刪除的功能，並且透過API的方式連接資料庫，對資料庫的資料做相對應的事情。
